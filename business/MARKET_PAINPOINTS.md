# 目标客户现状、痛点与 OASIS 的真实匹配度分析

## 一、他们现在怎么做投放预演

### 1.1 品牌方 / 甲方市场部

**现状：大部分品牌投放前没有真正的"预演"，靠的是经验+对标+赌。**

```
一次典型的投放决策流程：

1. 品牌经理定目标
   "这个月要推新品，预算 30 万，目标是小红书种草"

2. 找代理商/MCN 出方案
   代理商给一份 PPT：建议投 5 个腰部达人 + 200 个素人铺量
   依据是什么？→ "我们之前做过类似的案例"

3. 内部评审
   品牌总监问："为什么是 5 个腰部不是 10 个？为什么不投头部？"
   代理商答："根据经验..."（本质是拍脑袋）

4. 拍板执行
   选 A 方案还是 B 方案？→ 看谁的 PPT 做得好看、谁更能说

5. 投放执行
   投出去了，等数据回来

6. 事后复盘
   "这次互动率 3.2%，高于行业均值 2.5%，投放效果良好"
   （但没人知道如果换个策略是不是能到 5%）
```

**他们现有的"预演"手段**：

| 手段 | 怎么做 | 局限性 |
|------|--------|--------|
| **历史数据对标** | 看上次/竞品类似投放的数据 | 市场环境在变，历史不代表未来；竞品数据不完整 |
| **达人数据分析工具** | 用蝉妈妈/新榜/灰豚等工具看达人历史数据 | 只能看到达人过去的表现，不能预测这次内容的效果 |
| **小范围测试** | 先投一小波看数据，再决定放量 | 要花真金白银试；测试期也要 3-7 天；小样本数据不稳定 |
| **代理商经验** | 代理商/MCN 凭经验判断 | 主观偏见大；倾向于推荐利润高的方案 |
| **内部讨论** | 团队开会讨论、投票 | 本质是一群人拍脑袋 vs 一个人拍脑袋 |
| **焦点小组** | 找 8-12 个真人做小组访谈 | 贵（一场 2-5 万）、慢（2-3 周）、样本太小、存在社会期望偏差 |
| **问卷调研** | 发问卷收集潜在用户反馈 | 问卷回答 ≠ 真实行为（人们说的和做的不一样）|

### 1.2 MCN / 代理商

**现状：凭经验和数据工具选达人、定内容方向，但对"这条内容能不能火"基本是赌。**

```
MCN 接到品牌方 Brief 后的流程：

1. 拆解 Brief → 确定目标人群、内容调性
2. 从达人库里筛选匹配的达人 → 看粉丝画像、历史数据
3. 出内容方向建议 → 2-3 个方向供品牌方选择
4. 达人出稿/拍摄 → 品牌方审稿
5. 发布 → 等数据
6. 优化 → 根据第一波数据调整后续内容

"这条内容发出去用户会怎么反应？" → 没人知道，发了才知道
```

### 1.3 PR / 公关从业者

**现状：危机预案基本靠想象，舆情预判靠经验和直觉。**

```
典型的危机公关准备：

1. 列出可能的风险清单（"如果被曝出 XX 怎么办"）
2. 针对每个风险准备一套话术模板
3. 确定响应流程和发言人
4. 等危机真正发生再启动预案

问题：
- 预案是静态的，只考虑了"我们怎么回应"
- 没有考虑"回应之后公众会怎么反应"
- 更没有考虑"不同回应策略之间的效果差异"
```

## 二、他们的真实痛点

### 痛点清单（按严重程度排序）

#### 痛点 1：投放决策靠赌，试错成本高（最痛）

```
"30 万预算，选错了策略就是打水漂。"

具体表现：
├── A 方案和 B 方案都有道理，不知道选哪个
├── 代理商推荐的方案是不是最优的？无法验证
├── 每次都要花真钱去试，试错成本 = 一次测试投放的费用（通常是总预算的 10-20%）
└── 即使测试了，小样本数据波动大，结论不可靠

影响：
├── 保守派：反复选"安全"的方案，错过突破性创意
└── 激进派：靠直觉赌大的，赌输了就背锅

谁最痛：品牌市场经理（背 KPI 的人）
痛感等级：★★★★★
```

#### 痛点 2：不知道用户真实反应，只能看历史数据（很痛）

```
"我知道上次的数据是多少，但这次的内容不一样、市场环境不一样。"

具体表现：
├── 达人分析工具只能看历史表现，不能预测新内容
├── 内容质量无法量化评估——同一个达人，不同内容效果差 10 倍
├── 目标人群的真实态度只有投放后才知道
└── 竞品动作随时在变，历史数据的参考价值打折扣

影响：
├── 决策依据是"过去怎样"，不是"这次会怎样"
└── 内容创意评审变成主观审美的争论

谁最痛：品牌经理 + MCN 策划
痛感等级：★★★★☆
```

#### 痛点 3：效果归因不清，无法优化（很痛）

```
"这次效果好，但不知道好在哪里；下次效果差，也不知道差在哪里。"

具体表现：
├── 是内容好还是达人好？是时机好还是运气好？
├── 多个变量同时变，无法做控制变量实验
├── A/B 测试要在线上做，成本高且影响品牌一致性
└── 复盘报告大多是事后合理化，不是真正的因果分析

影响：
├── 经验无法有效积累——这次成功了不知道为什么，下次复制不了
└── 团队成长慢，一直在重复试错

谁最痛：品牌总监 + CMO
痛感等级：★★★★☆
```

#### 痛点 4：焦点小组和问卷调研又慢又贵（中痛）

```
"想做消费者调研，但一个焦点小组要 3-5 万，等结果要 2-3 周。"

具体表现：
├── 焦点小组：8-12 人，2-5 万/场，2-3 周出报告
├── 定量问卷：500 份以上才有统计意义，1-3 万，1-2 周
├── 频率限制：预算和时间限制导致一年只能做 2-4 次
├── 社会期望偏差：受访者说的 ≠ 真实行为
└── 样本代表性：8 个人能代表目标市场吗？

影响：
├── 大多数投放决策根本没有消费者调研支持
└── 有调研的项目也往往来不及等结果

谁最痛：市场研究/消费者洞察岗
痛感等级：★★★☆☆
```

#### 痛点 5：舆情爆发时没有预案演练（特定场景痛）

```
"危机来了手忙脚乱，预案写了但从没演练过。"

具体表现：
├── 预案是 Word 文档里的流程图，从没模拟执行过
├── 不同回应策略的效果差异没有量化评估
├── 不知道公众对不同表态的敏感度差异
└── 每次都是"边打边想"

影响：
├── 第一时间的回应往往不是最优的
└── 错过最佳回应窗口（前 2 小时）

谁最痛：公关总监 + 企业声誉管理
痛感等级：★★★☆☆（低频但高价值）
```

#### 痛点 6：代理商方案无法比价验证（隐性痛）

```
"代理商说这个方案最好，但我没有工具去质疑他。"

具体表现：
├── 品牌方和代理商存在信息差
├── 代理商倾向于推荐利润高（而非效果最好）的方案
├── 品牌方没有独立的评估工具
└── 只能事后看数据来评判，但这时候钱已经花了

影响：
├── 甲乙方信任成本高
└── 品牌方在谈判中处于弱势

谁最痛：品牌方采购/市场部
痛感等级：★★★☆☆
```

## 三、OASIS 能解决哪些痛点？老实说

### 能解决的 ✅

| 痛点 | OASIS 怎么解决 | 解决程度 |
|------|---------------|---------|
| **投放决策靠赌** | 在仿真环境中对比 A/B/C 多个方案，看哪个互动数据最好 | 70% — 不是精确预测，但比拍脑袋强得多 |
| **不知道用户真实反应** | 模拟 500 个不同画像用户的点赞/评论/转发行为和内容 | 60% — Agent 的评论内容能揭示用户关注点和态度 |
| **效果归因不清** | 控制变量实验——只改一个变量，其他不变，看差异 | 80% — 这是仿真的天然优势，现实中做不到 |
| **焦点小组太慢太贵** | 500 个 AI 用户 = 60 个焦点小组的样本量，2 小时出结果 | 50% — 不能替代真人调研，但能作为快速初筛 |
| **舆情预案没演练过** | 模拟负面事件爆发 + 不同回应策略，量化比较效果 | 70% — 可以模拟传播路径和情绪变化 |
| **代理商方案无法验证** | 把代理商的方案放进仿真里跑一遍，用数据质疑 | 60% — 给品牌方一个独立的评估工具 |

### 解决不了的 ❌（必须诚实面对）

| 局限性 | 原因 | 应对话术 |
|--------|------|---------|
| **不能精确预测具体数字** | AI Agent 不是真实用户，互动率/转化率的绝对值不准 | "我们提供的是策略对比和趋势判断，不是精确预测。就像风洞测试不能告诉你飞机能飞多远，但能告诉你哪个机翼设计更好" |
| **不能预测爆款** | 爆款有极大的随机性和运气成分 | "我们帮你排除明显会失败的方案，提高成功概率，但不保证爆款" |
| **不能替代真人调研** | AI 的反应模式 ≠ 真人反应模式 | "我们是调研的快速初筛工具，不是替代品。先用 AI 筛 10 个方案到 3 个，再用真人验证这 3 个" |
| **不能模拟平台算法** | 真实的小红书/抖音推荐算法是黑盒 | "我们模拟的是用户行为，不是平台算法。真实效果还会受平台流量分配影响" |
| **画像精确度受限** | Agent 画像再细致也不是真实用户 | "画像基于公开的用户研究数据，能代表大方向，但个体差异无法完全还原" |

## 四、核心定位：不要卖"预测"，卖"预演"

这是最关键的认知——**千万不要承诺"预测投放效果"，要承诺"帮你在投放前多想一步"**。

```
错误定位：
  "我们能预测你的投放 ROI"
  → 客户预期过高 → 实际效果对不上 → 口碑崩塌

正确定位：
  "我们帮你在投放前预演一轮，看看不同策略的差异"
  → 客户预期合理 → 实际效果超预期 → 口碑积累
```

### 类比话术

| 类比 | 说法 |
|------|------|
| **风洞测试** | "就像造飞机前在风洞里测试模型，不是预测飞多远，是对比哪个设计更好(像造飞机前的风洞测试，不是预测飞多远，是告诉你哪个机翼设计更好。)" |
| **沙盘推演** | "就像军事演习，不是预测战争结果，是提前暴露方案中的漏洞" |
| **彩排** | "就像演出前彩排一遍，不是预测观众反应，是让你更有信心上台" |
| **试衣间** | "就像买衣服前先试穿一下，不是预测你穿了多好看，是避免买错" |

## 五、针对每个人群的精准价值主张

### 对品牌市场经理

```
你的痛点：30 万预算选错策略就是打水漂，每次决策都像赌博
我的价值：在花钱之前，用 AI 模拟 500 个目标用户帮你"试投"一轮
你能得到：3 种策略的效果对比数据 + 最优方案建议
花多少钱：5000-8000 元（vs 真实测试投放的 3-5 万）
要多久：2 小时出结果（vs 测试投放的 3-7 天）
```

### 对 MCN / 代理商

```
你的痛点：给品牌方出方案时没有数据支撑，只能凭经验
我的价值：用仿真数据增强你的方案说服力，提高中标率
你能得到：方案附带仿真数据报告，让甲方看到"为什么选这个方案"
花多少钱：3000-5000 元（算进你给甲方的报价里）
你的收益：方案更有说服力 → 中标率提高 → 客单价可以提升
```

### 对 PR / 公关从业者

```
你的痛点：危机预案写了但从没演练过，不知道哪种回应最有效
我的价值：模拟危机爆发后 3 种回应策略的舆论走向
你能得到：每种策略的舆论发酵路径 + 情绪变化曲线 + 最优策略建议
花多少钱：1-3 万元
什么时候用：年度危机预案制定时 / 重大决策发布前
```

### 对市场研究/消费者洞察岗

```
你的痛点：焦点小组太慢太贵，一年只能做几次
我的价值：500 个 AI 用户快速初筛，2 小时出结果
你能得到：内容方向的快速验证 + 用户关注点分析
花多少钱：3000-5000 元（焦点小组的 1/10）
定位：不替代真人调研，做真人调研前的快速初筛
```

## 六、竞品/替代方案对比

客户不用你的工具，他们还能用什么？

| 替代方案 | 成本 | 时间 | 样本量 | 可控变量 | 定量分析 |
|---------|------|------|--------|---------|---------|
| **拍脑袋决策** | 0 | 即时 | 0 | 不可控 | 无 |
| **历史数据对标** | 0-免费工具 | 1-2 天 | 取决于数据 | 不可控 | 弱 |
| **蝉妈妈/新榜等工具** | 300-3000 元/月 | 即时 | 达人维度 | 不可控 | 中 |
| **小范围测试投放** | 3-10 万 | 3-7 天 | 真实但小 | 部分可控 | 强 |
| **焦点小组** | 2-5 万/场 | 2-3 周 | 8-12 人 | 部分可控 | 弱 |
| **定量问卷** | 1-3 万 | 1-2 周 | 500+ | 部分可控 | 强 |
| **OASIS 仿真** | 5000-8000 | 2-4 小时 | 500 Agent | **完全可控** | 强 |

**OASIS 的独特优势**：

1. **完全可控的变量**——只改内容/只换达人/只变策略，其他条件完全相同。这是真实投放和传统调研都做不到的。
2. **速度极快**——2 小时 vs 几天到几周。
3. **成本极低**——5000 元 vs 3-10 万。
4. **可重复**——对同一场景反复测试不同策略，零额外成本。

**OASIS 的劣势**：

1. **不是真实用户**——AI 模拟的反应与真人有偏差。
2. **绝对数值不可信**——互动率的绝对值没有参考意义，只有策略间的相对对比有意义。
3. **市场认知需要教育**——"AI 模拟用户"这个概念对大部分客户来说是新的。

## 七、最终判断：这个工具能不能卖出去

### 结论：能卖，但要注意定位

```
能卖的前提：
├── 1. 不承诺预测，承诺预演
├── 2. 不替代现有工具，做增量价值
├── 3. 价格定在"试一试不心疼"的区间（5000-8000）
├── 4. 第一批客户用效果说话，而不是用概念忽悠
└── 5. 强调"控制变量对比"这个独特价值，这是其他工具做不到的

卖不出去的情况：
├── 1. 承诺了精确预测，客户验证后发现不准 → 口碑崩
├── 2. 定价太高（>2 万），客户觉得不如拿这钱做真实测试
├── 3. 没有案例支撑，客户不信任 AI 模拟的可靠性
└── 4. 目标客户找错了——技术人员会质疑准确性，业务人员更容易接受
```

### 最适合的切入客户画像

```
最佳首批客户：
├── 行业：新消费品牌（美妆、食品、个护）
├── 规模：年营销预算 200-2000 万（太小没预算、太大有成熟体系）
├── 岗位：市场经理或品牌经理（直接背投放 KPI）
├── 心态：愿意尝试新方法、有一定的数据分析意识
└── 痛点：正在纠结两个投放方案选哪个

最不适合的首批客户：
├── 大型4A公司（流程长、决策慢、爱挑刺）
├── 传统行业品牌（数字化意识弱）
├── 纯技术背景的人（会质疑 AI 模拟的科学性）
└── 预算极低的小品牌（5000 元也嫌贵）
```

## 八、一句话回答三个问题

**他们现在怎么预演？**
> 大部分不预演，靠经验+历史数据+拍脑袋。少数做小范围测试投放（花真钱试）或焦点小组（贵且慢）。

**他们的核心痛点？**
> 投放决策像赌博——花出去的钱不知道效果好不好，两个方案不知道选哪个，选错了就是浪费预算。

**OASIS 能解决吗？**
> 能部分解决——不能预测精确数字，但能做控制变量对比实验，帮客户在多个方案中选出更优的那个。定位是"投放前的沙盘推演"，不是"投放效果的水晶球"。5000 元 + 2 小时的成本，比真实测试投放便宜 90%、快 10 倍。
