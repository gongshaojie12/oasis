# 自媒体内容体系规划

## 一、内容的终极目的

发内容不是为了涨粉，是为了**让潜在客户信任你，然后为你付钱**。

每一条内容都要回答潜在客户心中的一个问题：

```
阶段 1：这是什么？（认知）     → 让他知道"AI 仿真投放"这件事存在
阶段 2：这有用吗？（兴趣）     → 让他觉得"这东西好像真的能帮到我"
阶段 3：靠谱吗？（信任）       → 让他相信"这个人/团队做的东西是可信的"
阶段 4：多少钱？（转化）       → 让他觉得"这个价格值得试一试"
```

**不同类型的内容对应不同阶段，缺一个阶段客户都不会付钱。**

---

## 二、六大内容支柱

### 支柱 1：仿真实验（占 40%，核心引流内容）

**作用**：吸引注意力、建立专业感、提供谈资

**就是拿 OASIS 跑一个真实实验，把结果包装成内容。**

#### 选题方向 A：投放策略对比

用仿真对比不同营销策略的效果差异，给出数据结论。

```
选题示例：
├── "素人铺量 vs 腰部达人，同样预算哪个种草效果好？"
├── "同一个产品，感性文案 vs 成分党文案，用户更买账哪个？"
├── "早上 8 点发 vs 晚上 8 点发，互动数据差多少？"
├── "长文案 vs 短文案，小红书用户更愿意看哪种？"
├── "测评类笔记 vs 故事类笔记，哪种更容易引发讨论？"
└── "投 1 个 50 万粉达人 vs 投 10 个 5 万粉达人，传播链路有什么不同？"
```

#### 选题方向 B：用户反应观察

模拟一条具体内容发出后，不同画像用户的反应差异。

```
选题示例：
├── "同一条护肤品笔记，20 岁和 30 岁的用户关注点完全不同"
├── "男性用户看到美妆种草内容时，他们在想什么？"
├── "价格敏感型用户 vs 品质优先型用户，他们的评论差异"
├── "一线城市和三线城市用户对同一个品牌的态度差异"
└── "I 人和 E 人在社交媒体上的互动行为有什么不同？"
```

#### 选题方向 C：蹭热点实验

某个品牌/事件正在刷屏时，立刻做一个相关仿真。

```
选题示例：
├── "[某品牌]翻车后，3 种公关回应的用户反应模拟"
├── "[某新品]发布后，用户讨论的焦点是什么？AI 模拟了一下"
├── "如果[某品牌]的竞品现在发一条蹭热度的内容，用户会买账吗？"
└── "[某热搜事件]如果品牌方选择沉默 vs 回应，舆论走向有什么不同？"
```

#### 内容模板

```
标题：[数据/反差] + [悬念]
  例："500 个 AI 用户告诉我，你的种草策略可能选错了"

正文结构（800-1500 字）：
  1. 开头钩子（2-3 句）：结论先行，制造好奇
     "同样 10 万预算，选错投放策略可能让你白花 6 万。"

  2. 实验背景（100 字）：为什么做这个实验
     "最近帮一个品牌做投放方案时纠结了一个问题..."

  3. 实验设计（200 字）：模拟了什么、怎么设计的
     "我设置了 500 个不同画像的虚拟用户..."

  4. 核心发现（400-800 字）：3 个关键数据发现 + 图表
     发现一：[数据 + 解读]
     发现二：[数据 + 解读]
     发现三：[数据 + 解读]

  5. 实操建议（100 字）：对读者有什么用
     "如果你在做类似投放，建议..."

  6. 软植入（30 字）
     "以上实验用 AI 社交仿真完成，想试的可以私信我"
```

### 支柱 2：行业认知科普（占 20%，建立权威）

**作用**：让客户觉得"这个人很懂行"，建立专业信任

**不聊技术，聊品牌方日常面对的业务问题。**

#### 选题方向

```
投放决策类：
├── "为什么你的投放复盘总是'效果良好'，但下次还是靠猜？"
├── "品牌方选代理商方案时最常犯的 3 个错误"
├── "投放前的'测试投放'到底在测什么？大多数人理解错了"
├── "你以为的 A/B 测试 vs 真正的 A/B 测试"
└── "为什么同一个达人上个月爆了这个月不行？不是达人的问题"

用户洞察类：
├── "用户说'想买'和真的买，中间差了什么？"
├── "为什么焦点小组的结论经常翻车？"
├── "小红书用户的'收藏'到底意味着什么？"
├── "用户的'我会推荐给朋友'到底可不可信？"
└── "评论区的 100 条好评里，有多少是真心话？"

行业趋势类：
├── "AI 会怎么改变品牌投放？不是取代人，是减少赌博"
├── "为什么越来越多品牌开始重视'投前验证'？"
├── "从拍脑袋到数据驱动，中间还差一个'模拟实验'"
└── "海外品牌已经在用 AI 做营销仿真了，国内还没跟上"
```

#### 内容模板

```
标题：[痛点共鸣] 或 [反常识观点]
  例："你的投放复盘报告，其实什么都没复盘到"

正文结构（600-1000 字）：
  1. 痛点共鸣（3-5 句）：描述一个大家都经历过的场景
  2. 分析原因（200-400 字）：为什么会这样
  3. 解决思路（200-300 字）：应该怎么想这个问题
  4. 自然过渡到工具（50 字）：顺带提一下仿真的价值
```

### 支柱 3：案例展示（占 15%，促进转化）

**作用**：让客户看到"别人用了效果怎么样"，消除疑虑

**有真实客户后，这类内容的转化率最高。**

#### 早期（没有客户时）

用自己的实验作为案例：

```
├── "我帮朋友的护肤品牌做了一次投前模拟，她改了方案后互动率翻倍"
├── "一个真实案例：AI 仿真如何帮一个小品牌避开了一次公关危机"
└── "我用仿真帮自己的账号测试了 3 种选题方向，数据验证了哪种最好"
    （用自己的自媒体账号做实验，一举两得）
```

#### 有客户后

```
├── "[某行业]品牌用 AI 仿真优化投放策略，ROI 提升 XX%"
├── "客户反馈：'以前选方案靠开会，现在靠数据'"
├── "这个品牌在投放前做了一次模拟，避开了一个所有人都没想到的坑"
└── "30 分钟出报告 vs 以前等 2 周：一个 MCN 的使用体验"
```

#### 内容模板

```
标题：[客户角色] + [解决了什么问题] + [结果]
  例："一个美妆品牌用 AI 预演了投放方案，发现了一个所有人忽略的问题"

正文结构：
  1. 客户背景（谁、什么行业、面对什么决策）
  2. 原来怎么做的（痛点）
  3. 用仿真做了什么（过程，附截图）
  4. 发现了什么（关键洞察）
  5. 最终结果（客户的反馈/数据变化）
```

### 支柱 4：幕后揭秘（占 10%，拉近距离）

**作用**：展示真实的人和过程，增加信任感和亲近感

```
选题示例：
├── "我是怎么做一次仿真实验的？完整流程揭秘"
├── "这 500 个虚拟用户的画像是怎么设计出来的？"
├── "一次失败的实验：仿真结果和直觉完全相反，我重新跑了 3 遍"
├── "客户最常问我的 5 个问题"
├── "从技术 Demo 到第一个付费客户，我踩过的坑"
├── "一个人做 AI 营销工具的日常"
└── "这条内容发出前，我先用自己的工具模拟了一下你们的反应"
    （这条本身就是一个很好的营销）
```

**为什么要做这类内容**：

- 品牌方选服务商时，信任的是**人**，不是工具
- 展示你的思考过程，比展示结果更能建立深度信任
- 创业故事天然有吸引力和传播力

---

### 支柱 5：互动型内容（占 10%，增加粘性）

**作用**：引发讨论、收集需求、增加账号活跃度

```
选题示例：

投票/提问类：
├── "你在做投放决策时最头疼的是什么？"（评论区收集痛点）
├── "如果能在投放前知道一件事，你最想知道什么？"
├── "你觉得 AI 模拟 500 个用户的反应靠谱吗？"（引发讨论）
└── "下一期你想看我模拟哪个品牌/哪个场景？"（让用户决定选题）

免费服务类：
├── "评论区留下你最近要投放的品类，我挑 3 个免费做仿真"
├── "转发这条，抽 1 个人免费做一次 500 人仿真"
└── "这周开放 3 个免费体验名额，私信报名"
```

**为什么要做这类内容**：

- 评论区的讨论 = 免费的市场调研
- "免费做仿真"= 低成本获取潜在客户
- 互动数据好 → 平台推荐流量更大

---

### 支柱 6：方法论/工具分享（占 5%，SEO 长尾）

**作用**：搜索引擎长尾流量，吸引有数据分析能力的运营人员

```
选题示例：
├── "如何科学地做投放前的 A/B 测试？（不花冤枉钱版）"
├── "品牌投放效果归因的 3 种方法，最后一种最准"
├── "用 AI 做用户洞察的完整工作流"
├── "一个投放分析模板，帮你把复盘从'感觉良好'变成'数据说话'"
└── "我常用的营销数据分析工具清单"
```

## 三、各平台内容适配

同一个选题在不同平台呈现形式不同：

### 小红书（主阵地）

```
内容形态：图文笔记（4-6 张图 + 500-1000 字）
发布频率：每周 3-4 篇
内容比例：
  仿真实验  50%（图表截图 + 数据解读）
  行业认知  20%（痛点共鸣 + 短观点）
  互动型   20%（投票/免费体验/提问）
  幕后揭秘  10%

关键：
├── 首图必须有数据或对比图（在发现页吸引点击）
├── 标题要有数字（"500 个用户"、"3 种策略"、"效果差了 2 倍"）
├── 正文开头直接说结论（用户没耐心看铺垫）
└── 每篇结尾引导私信或关注
```

### 知乎（深度内容）

```
内容形态：长文（1500-3000 字）+ 回答行业问题
发布频率：每周 1-2 篇
内容比例：
  仿真实验  40%（深度实验分析）
  行业认知  30%（回答 "如何做好投放" 类问题）
  方法论   20%
  案例     10%

关键：
├── 去回答已有的高流量问题（"品牌营销怎么做"、"小红书投放"等）
├── 回答中自然引用自己的仿真数据
├── 长尾搜索流量，3-6 个月后持续带来曝光
└── 文末放个人介绍和联系方式
```

### 微信公众号（转化阵地）

```
内容形态：深度文章（1500-2500 字）
发布频率：每周 1 篇
内容比例：
  案例展示  30%（最能促进转化）
  仿真实验  30%
  行业认知  25%
  幕后揭秘  15%

关键：
├── 公众号是私域阵地，关注你的人更精准
├── 适合放完整案例和深度分析
├── 文末放服务介绍和联系方式
└── 用小红书/知乎引流到公众号
```

### 抖音/B站（视频内容）

```
内容形态：1-3 分钟短视频
发布频率：每周 1-2 条
内容比例：
  仿真实验  60%（录屏 + 解说最容易做）
  幕后揭秘  20%
  行业认知  20%

关键：
├── 录屏 + 配音解说，不需要露脸
├── 开头 3 秒必须抓住人（"同样 10 万预算，选错策略你可能白花 6 万"）
├── 视频中展示真实的仿真过程和数据变化
└── 视觉上看到数据在跑、评论在生成，比纯文字有冲击力
```

### 即刻（圈层传播）

```
内容形态：短帖（200-500 字）+ 图片
发布频率：每周 3-5 条
内容比例：
  仿真实验快报  40%（实验的核心发现，1-2 张图）
  行业思考     30%（短观点）
  互动讨论     20%
  创业日记     10%

关键：
├── 即刻用户偏创业者和早期采用者
├── 适合分享思考过程和创业进展
├── 讨论氛围好，容易获得高质量反馈
└── 即刻的"AI"和"营销"圈子都比较活跃
```

## 四、内容生产效率方案

**一个人做多个平台，必须极致提高效率。**

### 周内容生产流程

```
周一（2 小时）：跑一个仿真实验
  ├── 选题（30 分钟）：根据近期热点或积累的选题库选一个
  ├── 配置实验（30 分钟）：设置 Agent 画像和实验参数
  └── 运行 + 收集数据（1 小时）：跑仿真，导出数据，截图

周二（3 小时）：一次实验产出多平台内容
  ├── 小红书图文（1 小时）：4-6 张数据图 + 800 字
  ├── 知乎长文（1 小时）：补充分析，扩展到 2000 字
  ├── 即刻短帖（15 分钟）：核心发现 + 1 张图
  ├── 公众号文章（30 分钟）：知乎文章微调
  └── 抖音/B站脚本（15 分钟）：提炼 3 分钟解说词

周三-周五：发布 + 互动 + 准备下一个选题
  ├── 分平台定时发布
  ├── 回复评论（每天 15 分钟，评论区是转化的关键场所）
  └── 收集评论区的新选题灵感
```

### 选题库管理

维护一个选题库，灵感来了就记下来，不要到周一才想：

```
选题库来源：
├── 客户/潜在客户问过的问题
├── 行业社群里看到的讨论
├── 热点事件的延伸
├── 评论区读者的提问
├── 竞品做了什么新动作
└── 自己在做仿真时发现的有趣现象
```

## 五、内容节奏：不同阶段的侧重点

| 阶段 | 时间 | 内容重心 | 目的 |
|------|------|---------|------|
| 冷启动 | 第 1-4 周 | 80% 仿真实验 + 20% 行业认知 | 快速建立"这个人在做什么"的认知 |
| 积累期 | 第 2-3 月 | 50% 仿真实验 + 25% 行业认知 + 25% 互动 | 扩大曝光、收集潜在客户 |
| 转化期 | 第 4-6 月 | 40% 仿真实验 + 25% 案例 + 20% 行业认知 + 15% 互动 | 用案例推动付费转化 |
| 成熟期 | 第 7 月+ | 30% 案例 + 30% 仿真实验 + 20% 行业认知 + 20% 其他 | 案例驱动持续获客 |

## 六、什么内容不要做

| 不要做 | 原因 |
|--------|------|
| 纯技术分享（架构、代码、算法） | 目标客户看不懂，技术用户不会付钱 |
| AI 行业新闻转载 | 没有差异化，跟你的业务无关 |
| 心灵鸡汤/创业鸡血 | 不会带来客户 |
| 长篇产品介绍 | 没人看广告，要看案例和数据 |
| 追每一个热点 | 只追跟营销/品牌/投放相关的热点 |
| 日更低质内容 | 宁可少发，每篇都要有料 |

---

## 七、一句话总结

> 40% 的内容用仿真实验**吸引注意力**，
> 20% 的内容用行业认知**建立权威感**，
> 15% 的内容用真实案例**推动付费转化**，
> 剩下的内容用来**拉近距离和收集需求**。
> 每周跑一个实验，改写成 5 个平台的内容，总共花 5 小时。
